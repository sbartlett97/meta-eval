# requirements.txt -- Behavioral Invariance Eval Harness (PRD v3.1)
#
# This "core" set is everything needed to import the harness/ and judges/
# packages, run harness/evaluator.py + the judge panel, and run the test suite.
# It contains only pure-Python wheels that install cleanly on every platform, so
# `pip install -r requirements.txt` never fails on an unrelated heavy dependency.
#
# The heavy, platform-specific serving backend (vLLM) and the future
# meta-evaluation analysis stack are intentionally kept OUT of the default
# install and listed as clearly-marked optional extras below.
#
# Runtime quantization (BitsAndBytes) is intentionally NOT used: v3.1 serves
# pre-quantized checkpoints via vLLM / Ollama.

#   vLLM:   pip install "vllm>=0.3.0"
